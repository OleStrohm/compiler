CFG parse tree:
|                +               |
|     *    |          +          |
| num | id |     *    |     *    |
           | num | id | num | id |

Expansions:
    Expr => ( Expr )
    Expr => Sum
    Sum => Term + Sum
    Sum => Term
    Term => Factor * Term
    Term => Factor
    Factor => ( Expr )
    Factor => El
    El => num
    El => id
